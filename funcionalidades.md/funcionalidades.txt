# Funcionalidades do Projeto - Controle de Notas Escolares

Este projeto permite que o usuário controle notas de quatro bimestres, calcule a média e verifique se o aluno está aprovado ou reprovado.

## 1. Entrada de Notas
- O usuário insere as notas de **quatro bimestres** via console.
- As notas devem ser números entre **0 e 10**.
- O programa valida automaticamente a entrada e pede novamente caso seja inválida.
