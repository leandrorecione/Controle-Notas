# 📝 Controle de Notas Escolares

Este projeto é um **sistema simples de controle de notas escolares** em TypeScript.  
Ele permite que o usuário insira as notas de quatro bimestres, calcule a média e verifique se o aluno está **aprovado** ou **reprovado**.

---

## 🚀 Funcionalidades

- Inserir notas de **4 bimestres** via console
- Cálculo automático da **média**
- Verificação da **situação do aluno**
- Validação de notas (0 a 10)
- Saída clara com **notas, média e status**

---

## 🛠 Tecnologias Utilizadas

- TypeScript
- Node.js
- Readline (para interação via console)
